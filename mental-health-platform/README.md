# Mental Health Counseling Platform

## Overview
This project is a modern mental health counseling platform designed to provide users with a seamless experience in managing their mental wellness. The platform includes features such as mood tracking, session management, AI support, and access to counselors.

## Features
- **Dashboard**: A central hub for users to view their mood, upcoming sessions, AI support, and recent activities.
- **Chat**: A messaging interface for users to communicate with counselors and receive AI assistance.
- **Sessions**: A dedicated page for managing counseling sessions, including details about upcoming and past sessions.
- **Mood Tracker**: A feature that allows users to track their mood over time.
- **Counselors**: A page to explore available counselors, view their profiles, and book sessions.
- **Settings**: A section for users to manage their account settings.

## Technologies Used
- **React**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for styling the application.
- **TypeScript**: A superset of JavaScript that adds static types.

## Installation
To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd mental-health-platform
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

## Usage
- Navigate through the sidebar to access different sections of the platform.
- Use the dashboard to get an overview of your mental wellness.
- Book sessions with counselors and track your mood over time.
- Engage in chat for immediate support and assistance.
